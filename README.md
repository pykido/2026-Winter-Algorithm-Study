## 📝 Rules

### 1. 문제 풀이 (Daily Routine)
- **매일 2문제** (주말/공휴일 제외)
  - 필수 1문제 (해당 주차 유형) + 선택 1문제 (자율)
- **마감:** 당일 24:00 (자정)까지 PR 혹은 Push

### 2. 코드 리뷰
- **일정:** 주 2회
- **방식:** 사다리 타기로 발표자 선정
- **Review Point:** - 접근 방식 (Algorithm Mapping)
  - 시간복잡도 분석 ($O(N^2)$ vs $O(N \log N)$)
  - 코드 가독성 및 개선점 제안

### 3. Penalty
- **미제출:** 1문제당 **3,000원** (벌금은 모임 회식비/커피값으로 사용 ☕️)
